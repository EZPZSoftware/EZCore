# EZCore
EZCore is meant to be light, small, and an alternative to GNU Coreutils.


# Building
To build EZCore, run the following commands:

```
$ mkdir -v build
$ cd build
$ meson --prefix=/usr ..
$ ninja
$ ninja install
```
